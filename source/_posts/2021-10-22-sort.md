## 冒泡排序 (改进)

有的时候数组已经有序了，但一般的冒泡排序还会完整地进行 `n-1` 次比较。可以设置一个标志位，若发生了交换将其设为 true，否则设为 false。一轮比较结束后如果标志位还是 false 说明数组已经有序，不需要再比较了。

```java
public static void bubbleSortNew(int[] array) {
    boolean flag;
    for (int i = 0; i < array.length - 1; i++) {
        flag = false;
        for (int j = array.length - 1; j > i; j--) {
            if (array[j] < array[j - 1]) {
                int temp = array[j];
                array[j] = array[j - 1];
                array[j - 1] = temp;
                flag = true;
            }
        }
        if (!flag) {
			break;
        }
    }
}
```

## Shell 排序

Shell 排序又称缩小增量排序 (Diminishing Increment Sort)，根据某一增量将数组分为若干子数组，并对子数组分别进行插入排序。然后逐渐将增量减半，合并子数组，重复上述过程。当增量为1时，所有序列合并为一组，排序结束。

```java
public static void shellSort(int[] array) {
    int gap = array.length;
    while (true) {
        gap = gap / 2;
        for (int k = 0; k < gap; k++) {
            for (int i = k + gap; i < array.length; i += gap) {
                for (int j = i; j > k; j -= gap) {
                    if (array[j] < array[j - gap]) {
                        int temp = array[j];
                        array[j] = array[j - gap];
                        array[j - gap] = temp;
                    }
                    else {
                        break;
                    }
                }
            }
        }
        if (gap == 1) {
            break;
        }
    }
}
```

## 快速排序（递归）

基本思想：分而治之 (Divide-and-Conquer)。

* 从数列中取出一个数作为基准值；

* 将比基准值小的数全部放在基准值的左边，大于或等于基准值的数全部放在基准值的右边；

* 对左右两个子数组重复这个过程，直至各子数组只有1个数。

```java
public static void quickSort(int[] array, int left, int right) {
    if (left > right) {
        return;
    }
    int i = left, j = right;
    int base = array[left];  // 基准值
    while (i < j) {
        // 右边向左递减，寻找小于base的值
        while (i < j && array[j] >= base) {
            j--;
        }
        // 左边向右递增，寻找大于base的值
        while (i < j && array[i] <= base) {
            i++;
        }
        // 交换
        if (i < j) {
            int temp = array[i];
            array[i] = array[j];
            array[j] = temp;
        }
    }
    // 交换基准与i==j位置
    array[left] = array[i];
    array[i] = base;
    arrayPrint(array);
    // 左右两侧分而治之
    quickSort(array, left, i - 1);
    quickSort(array, j + 1, right);
}
```

## 归并排序（递归）

同样采用了分治法。归并排序自顶向下将数组分解到最小单元，再自底向上对这些单元进行按序合并处理。

```java
// 有序子数组合并
private static void merge(int[] a, int first, int middle, int last, int[] temp) {
    int i = first, j = middle + 1, k = 0;
    // 比较子数组元素大小，按序复制进辅助数组
    while (i < middle && j < last) {
        if (a[i] < a[j]) {
            temp[k++] = a[i++];
        }
        else {
            temp[k++] = a[j++];
        }
    }
    // 当其中一个子数组复制完毕时，将另一序列的全部剩余元素接在后面
    while (i <= first) {
        temp[k++] = a[i++];
    }
    while (j <= last) {
        temp[k++] = a[j++];
    }
    // 辅助数组内容覆盖进原数组
    for (int p = 0; p < k; p++) {
        a[first + p] = temp[p];
    }
}

// 数组归并排序
public static void mergeSort(int[] array, int first, int last, int[] temp) {
    if (first < last) {
        int middle = (first + last) / 2;
        mergeSort(array, first, middle, temp);
        mergeSort(array, middle + 1, last, temp);
        merge(array, first, middle, last, temp);
    }
}
```

## 堆排序

建立大/小顶堆，利用大/小顶堆的堆顶记录的是最大/最小关键字这一性质，向下调整得到升序/降序数组。

```java
// 调整根节点为i的数据，使其符合大顶堆的特征。i的子节点为2*i+1和2*i+2。
private static void maxHeapFix(int[] a, int i, int length) {
    int left, right;   // 子节点
    for (left = 2 * i + 1; left < length; left = 2 * i + 1) {
        right = left + 1;
        // 在左右子节点中寻找较大值
        int max = left;
        if (right < length && a[left] < a[right]) {
            max = right;
        }
        // 较大节点上移
        if (a[max] > a[i]) {
            int temp = a[i];
            a[i] = a[max];
            a[max] = temp;
        }
        else {
            break;  // 不需要调整
        }
    }
}

// 堆排序
public static void heapSort(int[] array) {
    // 将数组堆化（大顶堆）
    for (int i = (array.length - 1) / 2; i >= 0; i--) {
        maxHeapFix(array, i, array.length);
    }
    // 对堆化数组排序，最大值放在末尾
    for (int i = array.length - 1; i > 0; i--) {
        int temp = array[0];
        array[0] = array[i];
        array[i] = temp;
        maxHeapFix(array, 0, i);
    }
}
```

## 基数排序

基数排序（Radix sort）是一种非比较型整数排序算法。原理是将整数按位数切割成不同的数字，然后按每个位数分别比较。基数排序一般适用于正整数，存在0和负数时要做映射处理。

```java
public static void radixSort(int[] array) {
    int max = array[0]; // 最大值
    int min = array[0]; // 最小值
    for (int i : array) {
        if (i > max) {
            max = i;
        }
        if (i < min) {
            min = i;
        }
    }
    // 若最小值非正，则将所有数暂时减去(min-1)，以保证排序时所有数为正数
    if (min <= 0) {
        max -= (min - 1);
        for (int i = 0; i < array.length; i++) {
            array[i] -= (min - 1);
        }
    }
    // 遍历次数由数组最大值的位数决定
    for (int i = 1; max / i > 0; i = i * 10) {
        int[][] buckets = new int[array.length][10];
        // 获取每一位数字（个、十、百、千位...）按位进行桶排序
        for (int j = 0; j < array.length; j++) {
            int num = (array[j] / i) % 10;
            buckets[j][num] = array[j];
        }
        // 回收桶内元素
        int k = 0;
        for (int m = 0; m < 10; m++) {
            for (int n = 0; n < array.length; n++) {
                // 桶内有元素就回收
                if (buckets[n][m] != 0) {
                    array[k++] = buckets[n][m];
                    }
            }
        }
    }
    // 恢复原值
    if (min <= 0) {
        for (int i = 0; i < array.length; i++) {
            array[i] += (min - 1);
        }
    }
}
```
